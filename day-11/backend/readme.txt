* Four pillers of Authenticaion

1). Authenticaion
    -> identify karan ke request kis user ke pas se ai hai or kon kar raha hai
    -> User kon hai

2). Authorization 
    -> Ak user kya kya kar sakta hai
    -> Authority  of user means how much user have access
    

3). Validation
    -> Jo data hai uska formate check karna
    -> data ka formate check karna ke kay wo 
        formate sahi hai ya nahi

4). Varification
    -> Jo data hamare pas hi wo sahi hai ya nahi
    -> data sahi hai ya nahi
